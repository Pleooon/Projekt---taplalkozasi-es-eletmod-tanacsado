# 🏝️ A béke szigete – Táplálkozási és életmód tanácsadó weboldal

## 📌 Projekt leírás
Ez a weboldal egy modern, reszponzív platform, amely táplálkozási és életmód tanácsadást kínál.  
A cél egy olyan felhasználóbarát felület létrehozása, ahol a látogatók:
- megismerhetik a szolgáltatásokat  
- időpontot foglalhatnak  
- termékeket (ruházat) böngészhetnek  

---

## 🎯 Funkciók
- 🧑‍💼 Bemutatkozó oldal (Rólunk)  
- 📋 Szolgáltatások bemutatása  
- 📅 Időpontfoglalás felület  
- 🛒 Webshop (ruházat)  
- 📱 Reszponzív design (mobilbarát)  
- 🎨 Egyedi design Bootstrap segítségével  

---
